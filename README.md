# 🌤️ Mausam — Live Weather App

A simple and clean live weather web app built with pure **HTML, CSS, and JavaScript**.  
No frameworks. No paid APIs. Just the basics.

---

## 🌐 Live Demo
> Open `index.html` in your browser — that's it!

## ✨ Features

- 📍 **Live Location Weather** — Detects your current location automatically
- 🏙️ **Delhi Weather** — Always shows Delhi's weather alongside yours
- 🌡️ Temperature, Feels Like, Min & Max
- 💧 Humidity with visual progress bar
- 💨 Wind Speed & Direction
- 👁️ Visibility
- ☁️ Cloud Cover with visual progress bar
- 🌧️ Precipitation
- 🌅 Sunrise & 🌇 Sunset times
- 🌡️ Atmospheric Pressure
- 🔄 One-click Refresh button

| Technology | Usage |

---

## 🚀 How to Run

1. Clone or download this repository
2. Make sure all 3 files (`index.html`, `style.css`, `app.js`) are in the **same folder**
3. Open `index.html` in any modern browser
4. Allow **location permission** when prompted
5. Done! ✅

---

## 🔑 API Used

- **[Open-Meteo](https://open-meteo.com/)** — Free weather API, no API key required
- **[Nominatim](https://nominatim.org/)** — Free reverse geocoding by OpenStreetMap

> No hidden keys. No `.env` file needed. Works out of the box.
